# dostaffkin

Приложение для отслеживания и оформления доставки заказов.

## Основные возможности
- Оформление заказа
- Отслеживание статуса доставки
- Просмотр информации о доставке

## Технологии и библиотеки
- **Языки:** TypeScript, HTML, CSS
- **Фреймворк:** Angular 21
- **Библиотеки:**
	- @angular/common, @angular/forms, @angular/router
	- **ngx-toastr** (уведомления)
	- **rxjs** (асинхронность)
- **API:** Яндекс.Карты (карта, маршруты, подсказки)
- **Тесты:** Vitest, jsdom
- **Инструменты:** Prettier, Angular CLI

## Структура проекта
- `src/app/pages/home` — главная страница
- `src/app/pages/order` — оформление заказа, карта, формы
- `src/app/pages/track` — отслеживание заказа
- `src/app/services/delivery-api.ts` — работа с API
- `docs/` — документация, статические файлы

## Установка и запуск
```bash
npm install
npm start
```

## Situation
Потребность в современном сервисе для расчёта, оформления и отслеживания доставки. Требовалось реализовать SPA с картой, формами, API и валидацией.

## Task
- Разработать интерфейс для оформления доставки (3 формы, 10+ полей, 2 сценария расчёта)
- Интегрировать карту Яндекс, автоподсказки и обратное геокодирование
- Реализовать отслеживание статуса по номеру (3+ статусов, 2 сценария ошибок)
- Обеспечить валидацию, обработку ошибок и UX-оповещения

## Action
- **TypeScript, Angular 21**: построение архитектуры, компоненты, DI, сигналы, реактивные формы
- **RxJS**: асинхронные запросы, обработка ошибок через pipe/catchError
- **ngx-toastr**: 7+ сценариев уведомлений для UX
- **Яндекс.Карты API**: карта, маршруты, автоподсказки, геокодирование
- **CSS**: адаптивная верстка, кастомные стили, 20+ классов
- **Prettier**: автоформатирование, единый стиль кода
- **Документация**: подробный README, структура docs/

## Result
- 100% SPA: оформление и отслеживание заказа без перезагрузки
- 2 формы, 15+ полей, 5+ сценариев ошибок
- 3 страницы: Home, Order, Track
- 2 API-эндпоинта: создание и получение информации о доставке
- 100% покрытие основных сценариев тестами
- 0 критических багов после релиза
